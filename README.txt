Тестовое Задание для потенциальных PHP студентов
Книжный Каталог "BookBox"

Необходимые установки для целостной работы приложения:

1)Пропишите полный путь в файл .htaccess(bookbox\views\admin\.htaceess) к файлу .htpasswd для доступа к админ-панели.
	логин: admin   пароль: admin

2)Пропишите полный путь в файл addbook.php, updbook.php(bookbox\views\admin\addbook.php, bookbox\views\admin\updbook.php),
 для возможности добавления изображений в базу данных.

3)Создайте базу данных с именем "book_ctlg" затем импортируйте в нее файл book_ctlg.sql (данный файл прилагается к проекту).

4)Проект доступен на https://github.com/Dansleym/bookbox.git
